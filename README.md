## `cat repo.txt`

A structured collection of **Object-Oriented Programming concepts** implemented in **C++**, built during my 2nd semester at **UET Mardan**. This isn't copy-pasted code from tutorials ŌĆö every folder contains programs I wrote, debugged, and refactored to actually understand how OOP works under the hood.

&gt; **Note:** The repo name says "OPPS" because I was in a hurry when I created it. We're keeping it ŌĆö mistakes that ship are features.
